# ∞ Eternal Notes

**Заметки, которые живут вечно.** Зашифрованы. Неудаляемы.

Быстрая запись → E2E шифрование на клиенте → вечное хранение в блокчейне.

## Концепция

- **Zero friction** — открыл → написал → закрыл. Никаких папок, тегов, категорий
- **E2E шифрование** — AES-256-GCM, ключ из seed-фразы (BIP-39, 12 слов)
- **Восстановление** — ввёл 12 слов на новом устройстве → все заметки на месте
- **Вечное хранение** — Arweave (скоро), данные навсегда в блокчейне
- **AI поиск** — найди заметку живым языком (фаза 2)
- **Авто-связи** — AI находит связанные заметки (фаза 2)

## Запуск

```bash
npm install
npm run dev
```

## Стек

- **Frontend:** React + TypeScript + Vite
- **Шифрование:** Web Crypto API (AES-256-GCM), @scure/bip39
- **Хранение (MVP):** localStorage
- **Хранение (скоро):** Arweave через Irys

## Безопасность

- Ключ **никогда** не покидает устройство
- Seed-фраза — единственный способ восстановить данные
- Без seed → данные нечитаемы навсегда
